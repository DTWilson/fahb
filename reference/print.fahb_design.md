# Print a fahb design object

The default print method for a `fahb_design` object.

## Usage

``` r
# S3 method for class 'fahb_design'
print(x, coarse = TRUE, ...)
```

## Arguments

- x:

  object of class `fahb_design` as produced by
  [`fahb_design()`](https://dtwilson.github.io/fahb/reference/fahb_design.md).

- coarse:

  binary indicator that only a coarse subset of all decision rules
  should be printed. Defaults to TRUE.

- ...:

  further arguments passed to or from other methods.

## Value

no return value, called for side effects.
