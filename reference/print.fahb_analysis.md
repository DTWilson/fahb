# Print a `fahb` analysis object

The default print method for a `fahb_analysis` object.

## Usage

``` r
# S3 method for class 'fahb_analysis'
print(x, ...)
```

## Arguments

- x:

  object of class `fahb_analysis` as produced by
  [`fahb_analysis()`](https://dtwilson.github.io/fahb/reference/fahb_analysis.md).

- ...:

  further arguments passed to or from other methods.

## Value

no return value, called for side effects.
